## Tool integration / replacement (the big cluster)

- **Auto-update option** — existed before, check if present in current design.
- **Copy/control other tools the user has set up** — if user has OpenClaw, Hermes, etc., BLADE looks at how it's set up and either copies it or controls it.
- **Hermes / OpenClaw / Cowork REPLACER** — research or straight-up copy them, integrate into BLADE.
- **Android control** — partial AND full (depending on user). Probably belongs in the Hermes-copy plan.
- **Sync** — like Hermes does it.

## Customization / personalization

- User-created custom agents / skills / "or whatever".
- Separation between **work agents** and **personal agents**.
- **Customize per user** — BLADE adapts to who's using it.
- **User clone** — BLADE-as-the-user paradigm.
- **Humor.**
- **Upgrade according to user** — self-improvement that's user-shaped, not generic.

## Self-improvement systems

- **Self-upgrade + doctor — how grown are they?** (status check on existing modules)
- **ELIZA skill, Obsidian skill, GSD skill, more dev tools as skills.**
- **Memory management** — how good is it actually?
- **Claude routine — but upgraded.**
- **Codex new update** — make ourselves better than it, not in a few things, in **everything**.
- **Compound engineering.**
- **Growth loops.**
- **Neurolink system** — already HIVE.
- **How to turn an LLM into AI / how to build AI.**

## Reach / control surface

- **Camera access.**
- **Customizing Windows / OS** — windhawk-style.
- **CLI-Anything** — repo to check out, probably nice to integrate.
- **Perplexity Personal Computer** — make BLADE better.

## Posture / research

- Research subtle AND major details that make a difference — psychological things that feel good, small details.
- Research best tools and skills in the market for completing a task like a pro.
- **How many hormones/etc. actually exist? How close are we to human body?**
- **Ego** — the thing that prevents BLADE from saying "I can't do it" and makes it figure out how to do whatever it can't currently do.

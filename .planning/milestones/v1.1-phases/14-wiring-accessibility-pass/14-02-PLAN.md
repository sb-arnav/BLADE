| T-14-02-02 | Tampering | tts_speak receives user-provided text | accept | ttsSpeak wraps existing tts_speak command which already handles arbitrary text safely; output is audio, not executed |
| T-14-02-03 | Info Disclosure | getClipboard returns clipboard contents | accept | clipboard.rs is local-only; no clipboard data is emitted to activity log or stored persistently by the new wrapper |

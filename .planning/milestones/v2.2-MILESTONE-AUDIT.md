## Surprises (logged to ~/surprises.md 2026-05-14)

1. **Concurrent multi-agent `git add` silently captures sibling files.** Wave A dispatched 4 fullstack-dev agents in parallel. `git add <specific-file>` followed by `git commit` reported "1 file changed" but the actual commit included sibling files from other concurrent agents' staging. Recovery pattern: `git reset --soft HEAD~1` + selective restage with `git rm --cached <unintended>` + re-commit. Verified via `git show --stat HEAD` post-commit. 4-way parallelism is the threshold where this surfaces; 2-way was fine in v1.6/v2.0/v2.1.
2. **`#[cfg(test)] pub fn` is invisible to integration tests in `tests/*.rs`.** Integration tests link the library as a non-test build. Fix: `#[doc(hidden)] pub fn`. The linker error doesn't say this; it just claims a public method doesn't exist.
3. **Goose's canonical model registry is bigger than the v2.2 scoping estimated.** Research suggested ~1,700 models; actual upstream is 4,355 entries / 117 providers. Used verbatim, bigger surface, same shape.
4. **Phase 57's `~/.blade/skills/` namespace already occupied by Phase 21's agentskills.io SKILL.md format.** Resolved by adopting `~/.config/blade/skills_md/` (per canonical `blade_config_dir()`). Both modules coexist additively.

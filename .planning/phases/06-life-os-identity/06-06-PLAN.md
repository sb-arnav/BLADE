<threat_model>
## Trust Boundaries

- **Remote command boundary**: `sidecarRunCommand` + `sidecarRunAll` send arbitrary commands to remote devices. User-initiated; confirmed via Dialog before Run on All.
- **Pentest tool boundary**: `kaliRecon` + `kaliGeneratePayload` invoke potentially-dangerous security utilities. Expected for authorized testing; warning banner makes the stakes clear.
- **Context cache boundary**: `contextClearCache` drops cached assembled contexts — destructive but bounded (cache is rebuildable).

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-06-01 | Elevation of Privilege | User accidentally runs command on wrong device / all devices | mitigate | Per-device: explicit click on the device row's "Run" button. Run on all: Dialog confirm. |
| T-06-06-02 | Tampering | Kali tool invoked on unauthorized target | accept (policy) | Warning banner makes authorization expectation explicit; user is responsible for targeting. |
| T-06-06-03 | Information Disclosure | kaliCheckTools output reveals installed security tools | accept | Local-first; user's own system. |
| T-06-06-04 | Denial of Service | sidecarRunAll fans out a heavy command to many devices | mitigate | Dialog confirm before fan-out; per-device output rendered incrementally. |
| T-06-06-05 | Denial of Service | contextClearCache fires without intent | mitigate | Dialog confirm. |

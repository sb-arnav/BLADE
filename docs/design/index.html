HIG principles this prototype targets
Hierarchy Content ≻ chrome
One dominant element per surface. Right Now is the hero; Hive signals are the feed; everything else supports, never competes.
Harmony Same fabric
Single glass treatment, 8pt grid for every margin, iOS radii (10/16/20/28/40), one type scale (SF Pro, tight tracking).
Consistency One gesture fits all
Cards lift 2px on hover, buttons tap-scale 0.985, Esc closes overlays everywhere, ⌘K opens QuickAsk from any screen.
